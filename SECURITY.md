# Security Policy

## Supported Versions

The following versions of the ONEPIECE Framework currently receive security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2030    | :white_check_mark: |
| 2025    | :white_check_mark: |
| 2024    | :x:                |

Older versions are no longer supported.

## Reporting a Vulnerability

If you discover a security vulnerability, **please do NOT create a public issue**.

Instead, contact the project maintainer directly:

- Email: info@onepiece-framework.com
- Alternatively: https://github.com/onepiece-framework

We will respond as quickly as possible and work with you to verify and fix the issue.

## Disclosure Policy

- We follow responsible disclosure.
- Once the fix is ready, we will publish a security advisory.
